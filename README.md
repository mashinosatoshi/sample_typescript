# sample_typescript

## インストールしたもの
```
npm init

npm install  -D typescript @types/node

npm install -D eslint

npm install express

npm install -D @types/express
```

## Typescript初期化
```
tsc --init
```

## リポジトリ使用時の初期設定
```
npm install
```

## フォーマット
```
npm run lint
```

## ビルド
```
tsc
```

## 起動
```
node index.js
node hello.js
```
